## [1.0.0] - 2024-12-15

### Added
- Initial release of the CultureCoded JavaScript/TypeScript SDK
- `CultureCoded` client class with API key authentication
- `analyzeDesign()` - Create cultural UX analysis for designs
- `getAnalyses()` / `getAnalysis(id)` - Retrieve analysis results
- `deleteAnalysis(id)` - Delete an analysis
- `getEthnicGroups(country)` - Get cultural communities by country
- `getRegions()` - Get available target regions
- `exportAnalysis()` - Export analysis in PDF, CSV, or Figma formats
- `exportToFigma()` - Convenience method for Figma exports
- `getExports()` - List export history
- `getUser()` - Get current user profile
- `getUsage()` - Get API usage statistics
- Full TypeScript type definitions
- Comprehensive JSDoc documentation
